# Glitter lang

Glitter is an (in theory) backend-agnostic programming language, as of now primarily compiling to JavaScript.
It is inspired by C++ and JavaScript and it aims at combining their strengths, fixing their weaknesses and adding modern features.

It can be used for web development (client and server side), game development, and more (it is still experimental, but there's a lot more to come).

This here is an *experimental* implementation of a Glitter tokenizer, parser and compiler written in JavaScript.<br>
In the future it is planned to make a proper compiler in C++.
