# Glitter lang

Glitter is an (in theory) backend-agnostic programming language, as of now primarily compiling to JavaScript.
It is inspired by C++ and ECMAScript and it aims at combining their strengths, fixing their weaknesses and adding modern features.

It can be used for web development (client and server side), game development, and more (it is still experimental, but there's a lot more to come).

In this repository is an *experimental* implementation of a Glitter tokenizer, parser and compiler written in JavaScript.<br>
In the future it is planned to make a proper compiler in C++.

All the code is in `glitter.js`, and there are 0 dependencies (you can just use it right away).

Inside the `demo` folder you can find an interactive web playground (you can also try it [here](https://lstv.space/tech/glitter/playground/)) with various tools and examples.
